## Shadows & Daylight: Excalibur’s Labyrinth

A pixel-art exploration game built in HTML/JavaScript.

You play as Winter O’Brien, who wakes inside a sealed structure filled with locked rooms, artifacts, and environmental puzzles. Progress is driven by exploration, item use, and contextual interactions.

Based on the audio drama Shadows & Daylight more at [shadowsanddaylight.ca](https://shadowsanddaylight.ca)

* * *

## Controls

**Movement**

*   WASD — Move
    

**Interaction**

*   E — Interact / pick up items / activate objects
    
*   Space — Advance dialogue
    
*   Click dialogue box — Advance dialogue
    

**Inventory**

*   I — Open inventory
    
*   Click slot — Equip item
    
*   Only one item can be active at a time
    

**Gamepad support**

*   Left stick / D-pad — Move
    
*   A — Interact
    
*   X — Advance dialogue
    
*   Shoulder buttons / D-pad — Cycle inventory
    

* * *

## Core Systems

### Inventory

*   Items are collected from the environment and stored in slots
    
*   Only one item can be equipped at a time
    
*   Some interactions require a specific equipped item
    

### Dialogue System

*   Context-sensitive dialogue triggers based on:
    
    *   active item
        
    *   location
        
    *   story flags
        
*   Some objects contain multi-step dialogue sequences
